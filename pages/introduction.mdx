# Survey Platform

A full-stack **survey platform** for creating, publishing, and analyzing multi-language surveys with audience targeting, scoring, advice, and email notifications.

## What it does

- **Surveys** – Create surveys with multiple question types (single choice, multiple choice, open text, scale), pages, and optional scoring with score-based advice.
- **Audiences** – Define audiences and optionally restrict surveys to specific audiences (e.g. via secure token links).
- **Multi-language** – Translate survey content and UI; respondents choose a language at the start.
- **Public flow** – Share surveys via a public slug (`/public/:slug`) or secure token link (`/s/:token`). Optional contact step (name/email) before or after questions.
- **Responses** – View, filter, export, and optionally score responses; show advice based on score ranges.
- **Email** – Per-survey SMTP: admin notifications when a survey is completed, and optional respondent summary email (answers in their chosen language).
- **Analytics** – Overview, audience comparison, score distribution, and trends.
- **Translations** – Global UI strings and survey/question/option translations; optional DeepL integration for auto-translation.

## Tech stack

| Layer      | Technology |
|-----------|------------|
| Backend   | Laravel (PHP), Sanctum auth, SQLite/MySQL |
| Frontend  | React, React Router, TanStack Query, Tailwind CSS, Vite |
| API       | REST, JSON; public and protected (Bearer) routes |
